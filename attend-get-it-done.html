What is a Get it Done Session?

Get it Done sessions are live experiences taught by our expert instructors designed to help you prepare the required career artifacts.

Get it Done are live classes, 50 minutes in length designed to offer additional support and an opportunity for live Q&A. Get it Done sessions are offered seven days a week in the morning, afternoon, and evening.

Because Get it Done sessions are structured to focus on different aspects of career artifacts, you should come to the live session prepared. This means reading or watching the core competency content, and at least having reviewed the practice activity.

Get it Done sessions will help you complete the practice activity and the sprint challenge requirement.

Get it Done Preparation Checklist:

  • Did you review the core competency content before the live session?
  • Did you download and make a copy of the resume template?
  • Did you create your LinkedIn and GitHub accounts?
  • Did you read and attempt the Career Readiness practice activity?
