Welcome to ACS 2!

In this sprint, you will continue to improve upon your programming and problem-solving skills.

You are one step closer to being career ready! In this sprint, the career artifact focus is on building your GitHub profile.

Tips on how to succeed in this sprint:

  • Watch all the lessons and guided projects
  • Attend ACS Live Classes
  • Complete all practice activities in each module
  • Take the practice and official General Code Assessment (GCA) test
  • Build a complete GitHub profile

Review Introduction to CodeSignal and General Code Assessment (GCA) and Unit Syllabus before you begin.
